# Job-Portal

## How to run locally

Clone repo

```shell
git clone https://github.com/JayeshYadav07/Job-Portal.git
```

Start backend server code

```shell
cd /Backend
npm i

# Add .env values

node index.js
```

Start Frontend application

```shell
cd /Frontend
npm i
npm run dev
```

Check browser

```
http://localhost:5173/
```
